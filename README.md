# 데이터 분석 도구 🛠️📊

이 프로젝트는 NumPy와 Pandas를 활용한 데이터 분석 예제를 포함합니다.

## 📦 필요 라이브러리

- [NumPy](https://numpy.org/) 🥦
- [Pandas](https://pandas.pydata.org/) 📚

## 🔧 설치 방법

```bash
pip install numpy pandas
```
## 🐍 사용 예제
### NumPy 사용 예시
```
import numpy as np

# 배열 생성
arr = np.array([1, 2, 3, 4, 5])
print(arr)

```

### Pandas 사용 예시
```
import pandas as pd

# 데이터프레임 생성
data = {'이름': ['Alice', 'Bob', 'Charlie'],
        '나이': [24, 30, 22]}
df = pd.DataFrame(data)
print(df)

```
